用户中心系统

系统简介

用户中心系统是一个为网站或应用程序提供用户管理功能的关键组成部分。它包括用户注册、登录、个人资料管理、用户查询等基础功能，为用户提供了便捷的身份验证和个人信息管理途径。通过用户中心系统，用户可以轻松地注册账号，安全地登录网站或应用程序，并且管理其个人信息。

功能简介

用户中心系统包括以下等功能：

  1. 用户注册：允许新用户创建账户，填写必要的个人信息，并进行验证。
  2. 用户登录：提供安全的登录界面，允许已注册用户使用其凭据（如用户名和密码）登录系统。
  3. 个人资料管理：允许用户查看和编辑其个人资料信息，如姓名、电子邮件地址、联系方式等。
  4. 密码管理：允许用户更改密码、请求密码重置或找回密码等操作，以确保账户安全。
  5. 安全性：实施安全措施，如加密存储密码，保障用户数据的安全性。
  6. 权限管理：根据用户角色或权限级别，限制用户对系统资源的访问权限，确保信息安全和数据隐私。
  7. 用户体验优化：设计友好的用户界面和交互流程，提升用户体验，降低用户使用门槛，增强用户黏性和满意度。

技术栈与系统结构

前端

主要运用阿里 Ant Design 生态：

  • HTML + CSS + JavaScript 三件套‘
  • React 开发框架
  • Ant Desgin Pro 项目模板
  • Ant Design 端组件库
  • Umi 开发框架
  • Umi Request 请求库
  • 正向和反向代理

后端

  • Java 编程语言
  • Spring + Spring MVC+ Spring Boot 框架
  • MySQL 数据库
  • jUnit 单元测试库

部署

  • Linux 单机部署
  • Nginx Web 服务器
  • Docker 容器
  • 容器托管平台

数据库设计

数据库只有一张用户表，其中包含以下用户信息：

  1. id ：bigint，主键，用户id自增
  2. username：varchar，用户昵称
  3. userAccount：varchar，用户账号，登陆时使用
  4. avatarUrl：varchar，用户头像，上传图片URL路径即可
  5. gender：tinyint，性别
  6. userPassword：varchar，用户密码，通过MD5+盐值的方式对其进行加密
  7. phone：varchar，电话
  8. email：varchar，邮箱
  9. userStatus：int，用户状态，0为正常，如果用户存在异常操作，可将状态设置为1
  10. createTime：detetime，创建时间
  11. updateTime：detetime，更新时间
  12. isDelete：tinyint，逻辑删除，1为被删除
  13. userRole：int，用户角色，0为普通用户，1为管理员
